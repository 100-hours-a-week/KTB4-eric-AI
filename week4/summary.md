1. PyTorch

## 2. 데이터 전처리

데이터 전처리는 수집한 데이터를 분석이나 모델 학습에 적합하게 정리하는 과정입니다.

주요 과정은 다음과 같습니다.

1. **데이터 수집**  
   분석이나 모델 학습에 필요한 데이터를 모으는 과정입니다.

2. **결측값 처리**  
   비어 있는 값을 제거하거나 평균값, 중앙값 등으로 채우는 과정입니다.

3. **이상치 제거**  
   정상 범위를 벗어난 데이터를 확인하고 제거하거나 조정하는 과정입니다.

4. **데이터 정규화**  
   데이터의 값 범위를 일정하게 맞추어 모델이 안정적으로 학습할 수 있도록 하는 과정입니다.

5. **데이터 변환 및 증강**  
   데이터를 필요한 형태로 바꾸는 과정입니다.  
   이미지 데이터의 경우 이미지 회전, 이동, 확대와 같은 데이터 증강을 통해 학습 데이터를 늘릴 수 있습니다.

6. **데이터 분할**  
   전체 데이터를 학습용 데이터와 테스트용 데이터로 나누는 과정입니다.

---

## 3. 머신러닝

머신러닝은 컴퓨터가 데이터를 통해 규칙이나 패턴을 학습하고, 새로운 데이터에 대해 예측하거나 분류하는 기술입니다.

대표적인 머신러닝 알고리즘에는 랜덤 포레스트, K-NN, 나이브 베이즈 등이 있습니다.

### 랜덤 포레스트

랜덤 포레스트는 여러 개의 결정 트리를 만들고, 각각의 결과를 종합하여 더 안정적인 예측을 수행하는 알고리즘입니다.

### K-NN

K-NN은 새로운 데이터가 들어왔을 때 주변에 가장 가까운 데이터들을 보고 어떤 그룹에 속하는지 판단하는 알고리즘입니다.

### 나이브 베이즈

나이브 베이즈는 각 특징이 서로 독립적이라고 가정하고 확률을 계산하여, 가장 가능성이 높은 클래스를 예측하는 알고리즘입니다.

정리하면, 랜덤 포레스트, K-NN, 나이브 베이즈는 데이터를 기반으로 예측을 수행하기 위한 대표적인 머신러닝 알고리즘입니다.

---

## 4. 딥러닝

딥러닝은 인간의 뇌 신경망을 참고해 만든 인공 신경망을 여러 층으로 깊게 쌓아, 데이터의 복잡한 패턴을 학습하는 머신러닝 방법입니다.

### 퍼셉트론

퍼셉트론은 머신러닝의 가장 기본 단위로, 여러 입력값을 받아 출력값을 결정하는 간단한 인공 뉴런입니다.

### 시그모이드 함수

시그모이드 함수는 입력값을 0과 1 사이의 값으로 변환하는 활성화 함수입니다.  
모델이 단순한 계산 결과를 그대로 출력하지 않고, 분류나 확률 예측에 활용할 수 있도록 도와줍니다.

### 인공 신경망

인공 신경망은 여러 개의 퍼셉트론을 층처럼 연결한 구조입니다.  
보통 입력층, 은닉층, 출력층으로 구성됩니다.

### 완전 연결 신경망

완전 연결 신경망은 한 층의 모든 뉴런이 다음 층의 모든 뉴런과 연결된 구조입니다.  
이 구조를 여러 층으로 깊게 쌓으면 딥러닝 모델이 됩니다.

---

## 5. 딥러닝 학습 과정

딥러닝 모델은 예측값과 실제 정답의 차이를 줄이는 방향으로 학습합니다.  
이 과정에서 손실함수, 역전파, 옵티마이저가 사용됩니다.

### 손실함수

손실함수는 모델의 예측값이 실제 정답과 얼마나 차이 나는지 계산하는 함수입니다.  
즉, 모델이 얼마나 틀렸는지를 수치로 나타냅니다.

### 역전파

역전파는 출력층에서 입력층 방향으로 오차를 거꾸로 전달하면서, 각 가중치가 오차에 얼마나 영향을 주었는지 계산하는 과정입니다.

### 옵티마이저

옵티마이저는 역전파로 계산된 정보를 바탕으로 실제 가중치를 수정하는 알고리즘입니다.  
손실값이 줄어들도록 가중치를 업데이트합니다.
